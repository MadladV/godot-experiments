Aim Assist demo in Godot-Mono

Features various modes of aim assistance:

# 1. Slowdown
Camera rotation is slowed down when looking near an target

# 2. Follow
Camera follows the movements of the target on the screen

# 3. Snap-To
Camera is smoothly rotated towards the closest target

# 4. Bullet Magnetism
Shots are redirected towards the target
